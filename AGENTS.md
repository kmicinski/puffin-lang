This is a Git repository for a compilers course. This is the top-level
directory. You probably need to be looking in each subdirectory,
p1/2/3/4. This repository is a monorepo of multiple projects, each
written in the Racket language. Each project has a variety of ways to
test it. First, the main way is to call Racket and invoke main.rkt
with a suitable input file. There are several possible input files in
the corresponding test-programs/ directory. There is also test.rkt,
which is used by automated test scripts to run a batch of tests.
